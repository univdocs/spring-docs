# 13.6. 指标

Spring Boot执行器提供了依赖管理和Micrometer自动配置，Micrometer是一个应用程序指标外观，支持许多监控系统，包括：
+ AppOptics
+ Atlas
+ Datadog
+ Dynatrace
+ Elastic
+ Ganglia
+ Graphite
+ Humio
+ Influx
+ JMX
+ KairosDB
+ New Relic
+ Prometheus
+ SignalFx
+ Simple（内存的）
+ Stackdriver
+ StatsD
+ Wavefront

<univ-note type="tip">

要了解更多Micrometer功能，请参考Micrometer参考文档，特别是概念章节。

</univ-note>
